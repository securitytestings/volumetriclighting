# volumetriclighting
A module for Volumetric Lighting on ROBLOX. Includes heavy optimisation, and multiple options for higher performance.

Forked from a plugin by NOPESbucket, heavily optimised with new features. 

May include bugs. I'm updating it as much as possible.

Current Performance Stats:

CPU: Intel Xeon W2123
GPU: Nvidia GTX 1660 Super
RAM: 16GB

The average FPS is around 55, some resource intensive areas dropping down to 35-40, while some areas reach 80-100 FPS.
400-600MB Internal Roblox Memory Usage

These Performance Stats were using Max LOD (Beams + Billboards).
